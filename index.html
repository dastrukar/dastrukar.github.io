[[MODS]]

[Hideous Destructor Addons]

Hideous Helmet - An overhaul of sorts, this addon adds helmets and changes armour defense.

ZRE4PGHD - RE4/5-like item pickup glows. Technically works without HDest.

Stance Indicator - Generic stance indicator that you'd see in ARMA.

WIMP - What's In My Pack? A backpack UI overhaul.

[GZDoom Mods]

FDHUD - An "emulation" of FINAL DOOMER +'s HUD. Supports custom STBAR textures.

ggwp - Displays random messages and subtitles when you die. Also changes the music. I made this to feel better when I failed horribly in HDest :]

[[OTHER STUFF]]

dplay - A Doom launcher I made. Written in Python, it's meant to have a script like syntax; but I gave up improving it after getting tired with trying to compile Python.

dplayr - Another Doom launcher I made. Written in Rust, it's an improved version of dplay.

hddons-list - A website that lists a bunch of Hideous Destructor addons made by the community. (mainly the ones posted on the discord server)
